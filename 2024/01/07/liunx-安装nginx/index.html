liunx_安装 nginx

安装需要的依赖

sudo yum install -y wget gcc pcre pcre-devel zlib openssl openssl-devel

命令下载或者自己传上去

wget https://nginx.org/download/nginx-1.24.0.tar.gz

解压

tar -zxvf nginx-1.24.0.tar.gz

进入解压后的目录

cd nginx-1.24.0

配置编译选项

./configure –prefix=/usr/local/nginx –with-http_ssl_module

编译并安装Nginx

make && make install

启动Nginx服务

sudo /usr/local/nginx/sbin/nginx

在浏览器上输入ip看到nginx字样就是成功了
